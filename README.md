# 🤖 Rule-Based Chatbot

---

## 📌 Description

This project is a simple rule-based chatbot developed using Python. It responds to predefined user inputs such as greetings, common questions, and exit commands.

---

## ✨ Features

- Responds to greetings
- Answers predefined questions
- User-friendly console interaction
- Exit option using "bye"
- Easy to customize with new responses

---

## 🛠 Technologies Used

- Python 3

---

## ▶️ How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

```bash
python chatbot.py
```

---

## 💬 Sample Conversation

```
You: hello
Bot: Hi! How can I help you?

You: what is your name
Bot: I am a Rule-Based Chatbot.

You: bye
Bot: Goodbye!
```
